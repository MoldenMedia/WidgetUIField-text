WIDGET TUTORIAL: Widget UI Field - "text"

this widget descibes the usage of the widget UI field "text"

Example (you can copy and paste the following json into your UI definitions):

  • {
    • "id": "theTextField",
    • "type": "text",
    • "title": "Text",
    • "defaultValue": "Enter text into this line..."
  • }

Description of fields:

  • {
    • "id"............: id of the UI field
    • "type"..........: set to "text" to add a UI text field
    • "title".........: name of the UI field
    • "defaultValue"..: default value for the UI field when adding the widget to a compositions
  • }
